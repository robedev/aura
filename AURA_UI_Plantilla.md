
# 🎨 Plantilla UI/UX – Aplicación AURA

## 1. Principios de UI para AURA
- Interfaz no invasiva
- Elementos grandes y bien espaciados
- Confirmaciones claras
- Mínima carga cognitiva
- Todo configurable por perfil

---

## 2. Ventana principal de AURA

### Tamaño recomendado
- Ancho: 20–25% del ancho de pantalla
- Alto: 30–35% del alto de pantalla

### Posición por defecto
- Esquina inferior derecha
- Margen de seguridad: 32 px

### Comportamiento
- Siempre encima (opcional)
- Transparencia configurable (60–85%)
- Auto-ocultable
